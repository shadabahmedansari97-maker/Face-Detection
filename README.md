# Face Detection using YOLO 🎯

A real-time face detection system built using YOLOv5, 
trained on the WIDER Face dataset.

---

## 📌 What This Project Does
Detects human faces in images and video streams 
in real-time using the YOLO (You Only Look Once) 
deep learning architecture.

---

## 🛠️ Tech Stack
- Python 3.8+
- YOLO
- OpenCV
- PyTorch
- NumPy

---

## 📂 Dataset
**WIDER Face Dataset**  
- Source: http://shuoyang1213.me/WIDERFACE/
- Contains 32,203 images with 393,703 face annotations
- Used for training and evaluating the model

---

## ⚙️ How to Run

**1. Clone this repo**
git clone https://github.com/shadabahmedansari97-maker/face-mask-detection.git
cd face-mask-detection

**2. Install dependencies**
pip install -r requirements.txt

**3. Open the notebook**
jupyter notebook face_detection.ipynb
